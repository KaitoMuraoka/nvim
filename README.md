# Neovim設定

lazy.nvimを使用した最低限のNeovim開発環境。

## セットアップ

### 必要なツール

```bash
# Deno（skkeleton用）
brew install deno

# SKK辞書
mkdir -p ~/.skk
curl -o ~/.skk/SKK-JISYO.L.gz https://skk-dev.github.io/dict/SKK-JISYO.L.gz
gunzip ~/.skk/SKK-JISYO.L.gz
```

### インストール

Neovimを起動すると、lazy.nvimとプラグインが自動でインストールされる。

## キーバインド

リーダーキー: `<Space>`

### 基本操作

| キー | 説明 |
|------|------|
| `<Space>` + 待機 | which-keyでキーバインド一覧表示 |
| `<Space>e` | ファイルツリー開閉 |

### ファイル検索（Telescope）

| キー | 説明 |
|------|------|

| `<Space>fg` | 文字列検索（grep） |
| `<Space>fb` | バッファ一覧 |
| `<Space>fh` | ヘルプ検索 |

### LSP

| キー | 説明 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー情報 |
| `<Space>rn` | リネーム |

### コメント

| キー | 説明 |
|------|------|
| `gcc` | 行コメント切り替え |

### 補完（nvim-cmp）

| キー | 説明 |
|------|------|
| `<Tab>` | 次の候補 |
| `<S-Tab>` | 前の候補 |

## SKK（日本語入力）

Neovim内で動作する日本語入力。モード切り替え時のIME問題がない。

### 基本操作

| キー | 説明 |
|------|------|
| `<C-j>` | 日本語入力ON/OFF |

### 入力モード

| 操作 | 説明 |
|------|------|
| 小文字で入力 | ひらがな直接入力 |
| 大文字で開始 | 漢字変換モード |
| `q` | カタカナに変換 |
| `l` | ASCIIモードへ |
| `L` | 全角英数モードへ |

### 漢字変換の例

1. `<C-j>`で日本語入力ON
2. `Kanji`と入力（Kが大文字）→「▽かんじ」
3. `<Space>`で変換→「▼漢字」
4. `<CR>`または次の入力で確定

### 送り仮名の例

1. `OkuR`と入力（O, Rが大文字）→「▽おく*r」
2. `<Space>`で変換→「▼送r」
3. 確定すると「送り」

## Org-mode（メモ・タスク管理）

`.org`ファイルでメモやタスク管理ができる最低限のorg-mode機能を実装。

### 基本的な使い方

```org
* プロジェクト名

** TODO タスク1
タスクの説明

** DOING 作業中のタスク
   :LOGBOOK:
   CLOCK: [2026-01-19 Sun 10:00]--[2026-01-19 Sun 11:30] =>  1:30
   - 進捗メモ [2026-01-19 Sun 11:30]
   :END:

** DONE 完了したタスク
   :LOGBOOK:
   CLOCK: [2026-01-19 Sun 09:00]--[2026-01-19 Sun 10:00] =>  1:00
   :END:

*** サブタスク
- リスト項目1
- リスト項目2
```

### Clock In/Outの使い方

1. タスクの見出し上で `<Space>oI` を押すと作業開始（Clock In）
2. 作業中は自動的にタイムスタンプが記録されます
3. 作業終了時に `<Space>oO` を押すと作業時間が計算されます

### LOGBOOKメモの使い方（複数行対応）

`<Space>on` でタスクに関するメモを追加できます。

1. タスクの見出し上で `<Space>on` を押すと編集用のウィンドウが開きます
2. 複数行のメモを自由に書くことができます
3. `:wq` または `:x` で保存すると、メモがLOGBOOKに追加されます
4. `:q!` でキャンセルできます

メモは自動的にタイムスタンプ付きで記録され、複数行の場合は適切にインデントされます。

### TODO状態の管理

`<Space>ot` で状態を順番に切り替えます：
- 見出し → TODO → DOING → DONE → 見出し

### org-modeキーバインド

| キー | 説明 |
|------|------|
| `<Space>ot` | TODO/DOING/DONEの切り替え（見出し上で） |
| `<Space>oi` | 新しいTODOアイテムを挿入 |
| `<Space>oI` | Clock In（作業開始） |
| `<Space>oO` | Clock Out（作業終了） |
| `<Space>on` | LOGBOOKにメモを追加 |
| `<M-Left>` | 見出しレベルを上げる |
| `<M-Right>` | 見出しレベルを下げる |
| `<Tab>` | 見出しの折りたたみ切り替え |
| `<S-Tab>` | 全ての折りたたみ切り替え |

### サポートされている記法

- **見出し**: `*`, `**`, `***` など
- **TODO状態**: `TODO`, `DOING`, `DONE`
  - `TODO`: 未着手のタスク
  - `DOING`: 作業中のタスク
  - `DONE`: 完了したタスク
- **リスト**: `-`, `+`, `*` で開始
- **装飾**:
  - `*太字*`
  - `/イタリック/`
  - `_下線_`
  - `=コード=`
  - `~逐語~`
- **リンク**: `[[URL]]`
- **日付**: `[2026-01-20]`
- **タグ**: `:tag1:tag2:`
- **LOGBOOK**: タスクに紐づくメモと作業時間記録
  - Clock In/Outで自動的に作業時間を記録
  - メモをタイムスタンプ付きで記録
