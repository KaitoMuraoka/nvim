# Neovim設定

lazy.nvimを使用した最低限のNeovim開発環境。

## セットアップ

### 必要なツール

```bash
# Deno（skkeleton用）
brew install deno

# SKK辞書
mkdir -p ~/.skk
curl -o ~/.skk/SKK-JISYO.L.gz https://skk-dev.github.io/dict/SKK-JISYO.L.gz
gunzip ~/.skk/SKK-JISYO.L.gz
```

### インストール

Neovimを起動すると、lazy.nvimとプラグインが自動でインストールされる。

## SKK（日本語入力）

Neovim内で動作する日本語入力。モード切り替え時のIME問題がない。

### 基本操作

| キー | 説明 |
|------|------|
| `<C-j>` | 日本語入力ON/OFF |

### 入力モード

| 操作 | 説明 |
|------|------|
| 小文字で入力 | ひらがな直接入力 |
| 大文字で開始 | 漢字変換モード |
| `q` | カタカナに変換 |
| `l` | ASCIIモードへ |
| `L` | 全角英数モードへ |

### 漢字変換の例

1. `<C-j>`で日本語入力ON
2. `Kanji`と入力（Kが大文字）→「▽かんじ」
3. `<Space>`で変換→「▼漢字」
4. `<CR>`または次の入力で確定

### 送り仮名の例

1. `OkuR`と入力（O, Rが大文字）→「▽おく*r」
2. `<Space>`で変換→「▼送r」
3. 確定すると「送り」

## Org-mode（メモ・タスク管理）

[nvim-orgmode](https://github.com/nvim-orgmode/orgmode)を使用した高機能なメモ・タスク管理システム。

### セットアップ

初回起動時にプラグインが自動的にインストールされます。orgファイルは `~/org/` ディレクトリに配置してください。

```bash
mkdir -p ~/org
```

### 基本的な使い方

```org
* プロジェクト名

** TODO タスク1
タスクの説明

** DOING 作業中のタスク
   CLOCK: [2026-01-19 Sun 10:00]--[2026-01-19 Sun 11:30] =>  1:30

** DONE 完了したタスク
   CLOSED: [2026-01-19 Sun 12:00]

*** サブタスク
- [ ] チェックボックス付きタスク
- [X] 完了したタスク
```

### 主要機能

#### TODO管理
- **`<Space>ot`**: TODO状態を切り替え（TODO → DOING → DONE）
- **`<Space>oi`**: 新しいTODOアイテムを挿入
- **`<Space>o<Space>`**: チェックボックスをトグル

#### タイムトラッキング
- **`<Space>oI`**: Clock In（作業開始）
- **`<Space>oO`**: Clock Out（作業終了）
- 自動的に作業時間が計算されてLOGBOOKに記録されます

#### 日付・スケジュール
- **`<Space>od`**: タイムスタンプを挿入
- **`<Space>os`**: スケジュール日付を設定
- **`<Space>oD`**: 締切日を設定

#### タグとプライオリティ
- **`<Space>oT`**: タグを設定
- **`<Space>op`**: プライオリティを設定（[#A], [#B], [#C]）

#### 折りたたみ
- **`<Tab>`**: 見出しを折りたたみ/展開
- **`<S-Tab>`**: 全体を折りたたみ/展開

#### アジェンダビュー
- **`:OrgAgenda`**: アジェンダビューを開く
- スケジュールされたタスクや締切を一覧表示

#### その他
- **`<Space>oA`**: サブツリーをアーカイブ
- **`g?`**: ヘルプを表示

### TODO状態のカスタマイズ

TODO、DOING、DONEの3つの状態を使用し、それぞれ色分けされています：
- **TODO**: 赤色（未着手）
- **DOING**: 青色（作業中）
- **DONE**: 緑色（完了）

### サポートされている記法

- **見出し**: `*`, `**`, `***`
- **TODO状態**: `TODO`, `DOING`, `DONE`
- **チェックボックス**: `- [ ]`, `- [X]`
- **プライオリティ**: `[#A]`, `[#B]`, `[#C]`
- **タグ**: `:work:urgent:`
- **リンク**: `[[URL][説明]]` または `[[URL]]`
- **日付**: `<2026-01-20>`, `<2026-01-20 Mon>`
- **スケジュール**: `SCHEDULED: <2026-01-20>`
- **締切**: `DEADLINE: <2026-01-25>`
- **テーブル**: `| 列1 | 列2 |`
- **コードブロック**: `#+BEGIN_SRC ... #+END_SRC`
